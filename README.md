Tutorial notebooks providing a crash course through the basic picaso procedures needed to assess a target’s observability and science prospects given Roman-CGI's specifications. These draw heavily from the documentation maintained by the picaso team, which can be found here https://natashabatalha.github.io/picaso/tutorials.html#

To get started you can just click on notebook A_introduction_and_setup.ipynb within this github repository. It will walk you through installation either on your own machine or with Google Co Lab*. 

*Unfortunately Co Lab seems to be struggling with PICASO's SQL connection to the opacity database at this time, so you will need to do a local install for now. 

Accompanying reference data for this PICASO tutorial is here: https://zenodo.org/records/18892896 or can be downloaded from Arthur's hard drive.
