Tutorial notebooks providing a crash course through the basic picaso procedures needed to assess a target’s observability and science prospects given Roman-CGI's specifications. These draw heavily from the documentation maintained by the picaso team, which can be found here https://natashabatalha.github.io/picaso/tutorials.html#

To get started you can just click on notebook A_introduction_and_setup.ipynb within this github repository. It will walk you through installation either on your own machine or with Google Co Lab. Note that Co Lab seems to be struggling with the SQL connection to opacity database at this time. The embedded links within the tutorial's markdown will only render correctly in a local installation.

Accompanying reference data for PICASO is here: https://zenodo.org/records/18892896
